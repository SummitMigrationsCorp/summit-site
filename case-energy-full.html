Executive Summary

Exelon Corporation, one of America's largest energy utilities, required retirement of all legacy home directories and migration to OneDrive for Business. The effort demanded MNPI compliance, after-hours execution, dual-platform tooling, and zero tolerance for business impact.

Client Overview

Exelon operates nuclear, fossil, wind, and hydroelectric facilities nationwide. Tens of thousands of users stored data on aging NetApp systems with decades of permissions sprawl.

The Scale

45,000 home directories mapped and migrated, 120TB transferred, all executed after hours across a nine-month window.

Challenges

  • MNPI compliance requirements.
  • Decades of legacy permissions.
  • Multi-domain Active Directory complexity.
  • Network throttling risk requiring tight control.
  • Edge-case file types (Access DBs, OneNote, PSTs).

Spotlight: Dual-Platform Architecture

Summit deployed SkySync and Microsoft Migration Manager in parallel, enabling throughput, redundancy, and advanced handling of file types no single platform could manage alone.

Execution Methodology

  1. OneDrive provisioning
  2. Temporary admin access via PowerShell
  3. Nightly after-hours migrations with bandwidth tuning
  4. Morning validation before business hours
  5. Admin access revocation
  6. Legacy directory retirement

Tools & Technologies

  • SkySync
  • Microsoft Migration Manager
  • PowerShell automation
  • Microsoft Information Barriers

Results

  • 45,000 users moved with zero impact.
  • 120TB transferred with zero loss.
  • NetApp infrastructure fully retired.
  • All MNPI requirements satisfied.

Key Takeaways

  • Precision execution enables regulated environments to modernize safely.
  • Dual-tool strategy maximizes throughput and handles edge cases.
  • Nightly operations models eliminate business impact.
