Exelon – Large-Scale OneDrive Migration

A 45,000-user, 120TB file migration executed inside Exelon's nuclear-regulated Fortune 100 environment, completed with zero data loss and no disruption to operations.

Overview

Summit executed one of the largest and most compliance-sensitive OneDrive migrations on record, retiring decades-old home directories across Exelon's nuclear-regulated Fortune 100 environment without disrupting operations.

Challenges

  • Strict MNPI compliance in a regulated utility environment
  • Decades of legacy permissions requiring cleanup
  • Multi-domain identity complexity
  • Risk of network throttling requiring after-hours execution
  • Edge-case file types needing specialized handling

Spotlight: Dual-Platform Architecture

Summit deployed SkySync and Microsoft Migration Manager in parallel, enabling greater throughput, redundancy, and advanced handling of file types that no single platform could manage alone. All migrations ran after hours with progressive bandwidth tuning to protect business operations.

Our Approach

  • Dual-platform migration architecture using SkySync and Migration Manager
  • Nightly after-hours waves with strict bandwidth controls
  • Scripted OneDrive provisioning and temporary admin access management
  • Morning validation and immediate remediation before business hours

Results

  • 45,000 users migrated with zero data loss
  • 120 TB transferred across nine months
  • No business disruption across nuclear-regulated operations
  • Legacy NetApp infrastructure fully retired
