Client Overview

Knauf is one of Europe's most recognized manufacturing enterprises, with operations spanning more than 20 countries. Their file infrastructure consisted of hundreds of on‑premises servers distributed across multilingual, globally dispersed environments.

Challenges

  • Multilingual environments across a dozen regions
  • Offline facilities with zero internet connectivity
  • 400+ file servers across continents
  • Distributed decision‑makers across North America, Europe, and Asia

Spotlight: Solving Offline Migration

To migrate data from facilities with no internet connectivity, Summit deployed Microsoft Azure Data Box devices to each offline site. Data was copied locally, devices were shipped back to Microsoft for ingestion into Azure, and the migration was executed using standard tooling — ensuring every location was included in the global rollout.

Our Approach

  1. Global Discovery & Inventory
  2. Offline Remediation via Azure Data Box
  3. Country‑by‑Country Migrations with Migration Manager & ShareGate
  4. Full Reconciliation & Server Retirement per country

The Results

  • Complete retirement of 400+ servers
  • Zero data loss incidents
  • Offline facilities migrated successfully
  • Full reconciliation and sign‑off per region
