Overview

Summit Migrations serves as the dedicated M&A migration and governance partner for Edelman Financial Engines, one of the nation's largest independent wealth management firms. We manage every acquisition migration, from full tenant-to-tenant integrations to Record Center preparation, while continuously strengthening their Microsoft 365 security and compliance posture.

Challenges

  • Regulated environment (SEC and FINRA) requiring zero error tolerance
  • Each acquisition introduces a new M365 tenant and separate IT footprint
  • Client data must be prepared for ingestion into a Salesforce-integrated Record Center
  • Security posture needed proactive remediation between migrations

Our Approach

  • End-to-end M&A Tenant-to-Tenant migrations using ShareGate
  • Construction of the compliance Meta File for Record Center ingestion
  • Full security and governance advisory across external sharing, permissions hygiene, and admin access cleanup
  • Ongoing M365 platform administration and automation support

Results

  • All acquisition migrations completed with zero data loss
  • Record Center pipeline established and executed for all client data
  • Tenant-wide external sharing remediation and policy overhaul
  • Unauthorized Site Collection Admin access eliminated
  • Legacy on-premises home folders fully retired
